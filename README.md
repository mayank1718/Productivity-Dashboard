# 🚀 Ultimate Productivity Dashboard

An all-in-one productivity dashboard designed to boost your daily efficiency 📈 and help you stay organized. This project combines multiple useful tools into a single clean and interactive interface.

---

## ✨ Features

* ✅ **To-Do List** – Add, update, and manage daily tasks
* 🌦️ **Weather Updates** – Real-time weather information
* ⏰ **Live Clock** – Current date and time display
* 🗓️ **Daily Planner** – Plan your day effectively
* 💡 **Motivational Quotes** – Stay inspired throughout the day
* 🍅 **Pomodoro Timer** – Improve focus using time intervals
* 🌙 **Dark/Light Mode** – Personalized theme experience
* 📊 **Progress Tracking** – Track your task completion

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* APIs (for weather & quotes)
